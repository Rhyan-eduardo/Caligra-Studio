    CSS e JS compilados
    


    Baixe o código compilado pronto para uso para bootstrap v4.1.3 para entrar facilmente no seu projeto, o que inclui:
    


      
  
    • Pacotes CSS compilados e minizados (ver comparação de arquivos CSS)
      • 
  
    • Plugins JavaScript compilados e minizados
      • 

    


    Isso não inclui documentação, arquivos de origem ou quaisquer dependências javascript opcionais (jQuery e Popper.js).
    


    Download
    


    Arquivos de origem
    


    Compile bootstrap com seu próprio pipeline de ativos baixando nossos arquivos de sass, JavaScript e documentação de origem. Esta opção requer algumas ferramentas adicionais:
    


      
  
    • Compilador sass (Libsass ou Ruby Sass é suportado) para compilar seu CSS.
      • 
  
    • Prefixor automático para prefixação do fornecedor CSS
      • 

    


    Se você precisar de ferramentas de construção,elas estão incluídas para o desenvolvimento do Bootstrap e seus docs, mas provavelmente são inadequadas para seus próprios propósitos.
    


    Fonte de download
    


    BootstrapCDN
    


    Pule o download com bootstrapCDN para fornecer a versão em cache do CSS e JS compilados do Bootstrap para o seu projeto.
    


    


    Se você estiver usando nosso JavaScript compilado, não se esqueça de incluir versões de CDN de jQuery e Popper.js antes dele.
    


    


    Gerentes de pacotes
    


    Puxe os arquivos de origem do Bootstrap para quase qualquer projeto com alguns dos gerentes de pacotes mais populares. Não importa o gerenciador de pacotes, o Bootstrap exigirá um compilador Sass e um Autoprefixer  para uma configuração que corresponda às nossas versões compiladas oficiais.
